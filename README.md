# BookNex

BookNex is a premium sports venue booking application that allows users to discover, book, and manage sports venues like turfs, courts, and fields.

## Features

- **User Dashboard**: Discover venues, book slots, manage bookings.
- **Venue Owner Portal**: List venues, manage pricing, view reports, and handle bookings.
- **Admin Portal**: Approve venues, manage users, and broadcast announcements.
- **Real-time Notifications**: Web Push and in-app notifications.
- **PWA Support**: Installable on mobile devices with offline capabilities.

## Tech Stack

- **Frontend**: React, TypeScript, Vite, Tailwind CSS, shadcn-ui
- **Backend**: Supabase (PostgreSQL, Auth, Realtime, Edge Functions)
- **Deployment**: Vercel

## Getting Started

1.  Clone the repository.
2.  Install dependencies:
    ```sh
    npm install
    ```
3.  Run the development server:
    ```sh
    npm run dev
    ```

## Deployment

The project is configured for deployment on Vercel. Pushing to the `main` branch triggers an automatic deployment.
